# CookPlanner AI v1.3

## 更新方法

1. ZIPをすべて展開します。
2. 展開したフォルダ内のファイルを、現在使用中の `CookPlannerAI` フォルダへコピーします。
3. 同名ファイルは置き換えます。
4. 現在の `data` フォルダは削除しません。このZIPには利用者データを同梱していません。
5. Pythonを停止してから再起動します。

```powershell
taskkill /F /IM python.exe
python -m streamlit run app.py
```

## 新しい使い方

冷蔵庫画面で現実の残量に修正した後、`この在庫で献立を立て直す` を押してください。
今日より前の献立は残り、今日以降の献立と買い物リストだけが更新されます。
