# CookPlanner AI v1.4.3

## 追加機能

- ChatGPTへ渡す出力形式をJSONへ固定
- 貼り付けた回答から主菜・副菜・汁物を自動解析
- 解析結果を画面で確認してから採用可能
- 採用した献立を指定日の献立へ登録
- 新しい料理をレシピへ自動保存
- 不足食材を現在の買い物リストへ追加・統合
- APIキー不要、追加料金なし

## 更新方法

現在のCookPlannerAIフォルダへ全ファイルを上書きし、dataフォルダは既存データを残してください。

```powershell
python -m streamlit run app.py
```

## テスト手順

1. AIシェフでプロンプトをコピー
2. ChatGPTへ貼り付ける
3. JSON回答をアプリへ貼り戻す
4. 「回答を読み取る」
5. 内容を確認
6. 「献立・レシピ・買い物へ反映」

冷蔵庫の残量予測は表示のみです。冷蔵庫自動更新は次版で実装します。
