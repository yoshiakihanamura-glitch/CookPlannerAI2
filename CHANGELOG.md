# Changelog

## v1.4.1 - 無料AIブリッジ

- OpenAI API連携、APIキー入力、API課金を削除
- ChatGPT向けプロンプト自動生成を追加
- 冷蔵庫、残量、期限、妊娠情報、直近14日献立など20条件を自動反映
- ブラウザのクリップボードへコピーするボタンを追加

- 新規料理をレシピへ自動保存
- 不足食材を買い物リストへ追加・統合
