# CookPlanner AI v0.8

花村家の献立・冷蔵庫・買い物をまとめて管理するStreamlitアプリです。

## 起動方法

```bash
python -m streamlit run app.py
```

## v0.8の主な機能

- レシピ登録・編集・削除
- 30日分の献立作成
- 今日・明日の献立表示
- 冷蔵庫の残量・期限管理
- 献立の材料を食材名と単位ごとに集約
- 冷蔵庫の在庫を差し引いた不足分の買い物リスト
- 購入済みチェック
- 購入済み食材を冷蔵庫へ追加
- ホームに冷蔵庫・期限・買い物・レシピ件数を表示

既存の `data` フォルダは上書きせず、そのまま利用してください。
